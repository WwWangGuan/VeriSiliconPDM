◆ osal_lock_mutex()

int osal_lock_mutex ( OsalMutex *  mu,
uint32_t  timeout 
)

Lock or take a mutex.

Parameters
muThe mutex to be locked
timeoutThe maximum amount of time the task should remain in the blocked state to wait for the mutex if it is not already available, in millisecond. 1) If timeout is zero, then osal_lock_mutex will return immediately if the mutex is not available. 2) Setting timeout to OSAL_WAIT_FOREVER will cause the task to wait indefinitely (without a timeout)
Returns
int OSAL_TRUE for success, others for failure
