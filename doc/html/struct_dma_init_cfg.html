Detailed Description

This structure is used contains parameters that must be initialized for the transfer.

Note
: For src_addr and dst_addr which is memory type need to use non-cacheable RAM for the SoC who has no DLM
: If cyclic mode is used, ensure that each block size is at least 128 bytes in order for block interrupts to be normal
