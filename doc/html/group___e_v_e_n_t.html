int osal_wait_event ( void *  queue,
void **  data,
uint32_t  timeout 
)

Wait for event data from the specific event queue.

Parameters
queueEvent queue handle
dataEvent data that needs to be received
timeoutThe maximum amount of time the task should remain in the blocked state to wait for data to become available on the queue, in milliseconds. 1) If timeout is zero, then osal_wait_event will return immediately if the queue is already empty 2) Setting timeout to OSAL_WAIT_FOREVER will cause the task to wait indefinitely (without a timeout)
Returns
int Event ID is for success, EVENT_INVALID is for failure
