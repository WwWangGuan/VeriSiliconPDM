uint16_t DmacXferCfg::dst_sct_cnt

These parameters are used to config the scatter feature of SNPS AHB DMAC. if you don't what you are doing, please leave these arguments with 0. dst_sct_cnt: Destination contiguous transfer count between successive scatter boundaries. dst_sct_intvl: interval of contiguous destination scatter block. Note: corresponding to DSRx Register
